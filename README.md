## Publish

1. Merge your PR into `master`.

2. Add a new version
  ```sh
  npm version major|minor|patch -m "Version %s - INSERT_REASONS"
  ```

3. Publish
  ```sh
  npm publish
  ```

4. Push commit with tags
  ```sh
  git push origin master --tags
  ```
